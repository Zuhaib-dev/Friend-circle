## 🌍 // ABOUT THE NETWORK
Friend Circle V2 is the ultimate digital campfire and command center. It’s where the crew logs their expeditions, tracks off-road coordinates, uploads tactical intel, and organizes global operations (or just local fishing trips).

Designed with a **Brutalist / Spy-Tech** aesthetic: Monospace typography, hairline borders, terminal interfaces, and high-contrast monochrome design.

---

## 🛠️ // TECHNICAL ARCHITECTURE
- **Core Engine:** Next.js 16 (App Router)
- **Database:** MongoDB (Mongoose ORM)
- **Authentication:** NextAuth.js (Secure encrypted sessions)
- **Styling:** Tailwind CSS (Custom brutalist tokens)
- **Animations:** Motion / Framer Motion
- **Asset Storage:** ImageKit (Cloud Image/Video Processing)

---

## 📂 // ACTIVE MODULES

### 1. 🛡️ ADMIN DISPATCH CONSOLE
The nerve center of the application. 
- **User Management:** Promote/demote field agents, adjust clearances, and purge rogues.
- **Intel Feed Management:** Monitor, redact, or delete posts from the global network.
- **Tour Itineraries:** Draft highly detailed operational parameters (coordinates, elevation, distance) for upcoming adventures.

### 2. 🗺️ TACTICAL TOURS
A public-facing dispatch log detailing past and future operations. 
- Features military-grade readouts of mission parameters (ELEV, DIST, PARTY SIZE).
- Includes an encrypted comms channel (comment section) for field agents to coordinate drops.

### 3. 📸 INTEL ARCHIVE (GALLERY)
A massive, high-performance visual database.
- Seamlessly merges legacy V1 archives (200+ localized frames) with the live MongoDB intel feed.
- Powered by Intersection Observers for infinite scrolling.
- Instant LATEST/OLDEST/SIZE sorting and instant search filtering.

### 4. 👥 CREW REGISTRY
An interactive, heavily animated roster of all active personnel with micro-interactions, haptic feedback UI, and live status blinking dots.

## 🚀 // DEPLOYMENT INSTRUCTIONS

To spin up a local instance of the command center:

```bash
# 1. Navigate to the V2 directory
cd v2

# 2. Install dependencies
npm install

# 3. Setup Environment Variables
# Create a .env file and add your MongoDB URI, NextAuth secret, and ImageKit keys.

# 4. Boot the server
npm run dev
```

Navigate your terminal to `http://localhost:3000` to access the mainframe.
